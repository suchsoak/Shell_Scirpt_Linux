# Shell_Scirpt_Linux
This is a shell script for you to have everything when formatting your linux

This script is under development

# Ubuntu | Debian

## Install

> with Git

```sh
  git clone https://github.com/suchsoak/Shell_Scirpt_Linux.git
```

## Use

```sh
  chmod +x scritp.sh
```

```sh
  bash ./scritp.sh
```
